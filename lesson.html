AIGC 与信息技术

本课程介绍人工智能生成内容（AIGC）的基本概念、关键技术与应用场景，涵盖文本、图像、音频与代码生成技术。

主要内容：
  • AIGC 概览与发展趋势
  • 生成模型基础（语言模型、扩散模型等）
  • 主流工具与平台实操
  • 伦理、隐私与应用案例分析

网页设计与制作（Web 前端设计）

面向现代前端开发的实践课程，覆盖 HTML、CSS、JavaScript 基础与响应式设计，带你从静态页面到交互式网站的实现。

主要内容：
  • HTML 语义化与可访问性
  • CSS 布局、Flexbox 与 Grid
  • JavaScript 与 DOM 操作
  • 小型项目：个人主页或课程展示页

Python 程序设计

本课程从编程基础出发，使用 Python 语言教授算法思维、数据结构与常用库的使用，适合初学者与希望快速上手数据与自动化开发的同学。

主要内容：
  • 语法基础、控制结构与函数
  • 常用数据结构（列表、字典、集合）
  • 文件操作、异常处理与模块化编程
  • 入门数据分析与简单项目实践

人工智能（AI）

介绍人工智能与深度学习的基础概念、常用模型与应用，包括机器学习流程、神经网络基础与实际案例分析。

主要内容：
  • 机器学习基础与评估方法
  • 神经网络与深度学习概念
  • 常见模型（CNN、RNN、Transformer）简介
  • 项目实战：小型图像/文本任务
